## develop

### install libraries

```
$ yarn install
```

### test

```
$ yarn run test
$ yarn run test-watch
$ yarn run test-init // expect.jsを消してから実行すると自動生成される
```

### build

```
$ yarn build
```

### publish

```
$ yarn run publish-all
```
